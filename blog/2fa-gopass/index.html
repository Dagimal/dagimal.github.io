2fa dengan Gopass

05 Nov, 2025

Udah lama ga buka github, tiba-tiba kemarin waktu buka di ingetin suruh aktivasi 2fa buat login ke GitHub, yaudah tanpa mikir panjang langsung gas eksekusi pake gopass aja karena ini password manager yang udah integrasi fitur otp.
Gopass - An awesome password manager for awesome teams

Buat Credentials

Masukkan secret key 2fa otp.

gopass insert github/dagimal

Cek Credentials

gopass show github/dagimal

akan tampil seperti ini :

Generate OTP

gopass otp github/dagimal

gopass akan generate otp :
